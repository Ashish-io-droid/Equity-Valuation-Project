# Eternal Limited Valuation Project

Professional equity valuation project for Eternal Limited, formerly Zomato.

## Objective

Build an interview-defensible valuation project resembling first-year investment banking / equity research work:

- Excel financial model
- Equity research report
- Investment pitch deck
- Documented source pack and methodology

## Current Version

`v1.1 Professional Structure`

Completed:

- Numbered workbook tabs
- Raw data staging layer
- Dashboard
- Source audit structure
- DCF, trading comps, precedents, football field, thesis, and checks framework
- Portfolio-style project folder structure

## Folder Structure

```text
excel/                  Final working Excel model
data/                   Annual reports, quarterly results, investor presentations
docs/                   Methodology, model guide, source list
images/                 Screenshots for README/report/deck
outputs/                Versioned model exports and QA previews
presentation/           Investment pitch deck
reports/                Equity research report
```

## Workbook Flow

```text
Source filings
-> 03_Raw_Data
-> 04_Historical_FS
-> 06_Operating_Model
-> 08_DCF / 09_Trading_Comps / 10_Precedents
-> 11_Football_Field
-> 02_Dashboard / 12_Thesis
```
