# Source List

Primary source tracking lives in workbook tab `14_Sources_Audit`.

## Initial Source Categories

- Eternal investor relations
- Annual reports
- Quarterly results
- Investor presentations
- Exchange filings
- Risk-free rate source
- Equity risk premium source
- Beta / market data source
- Peer company filings
- Transaction announcements and filings

## Source Discipline

Every historical input should have:

- Source ID
- Period or as-of date
- Units
- URL or file name
- Notes explaining any conversion or normalization
